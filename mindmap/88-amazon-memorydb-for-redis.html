Mind Map: Amazon MemoryDB for Redis

AWS Redis-compatible In-memory Database

Amazon MemoryDB
for Redis

Định nghĩa

Amazon MemoryDB for Redis là dịch vụ cơ sở dữ liệu in-memory tương thích Redis, bền vững (durable) được quản lý hoàn toàn bởi AWS.

  • Redis-compatible, durable, in-memory database service
  • Là database thực sự với Redis-compatible API
  • Cung cấp dữ liệu in-memory nhưng với tính bền vững (durable data storage)
  • Multi-AZ transaction log để đảm bảo data durability
  • Được quản lý hoàn toàn bởi AWS

Khác biệt với Redis

So sánh MemoryDB vs Redis

Khía cạnh Redis (ElastiCache) MemoryDB for Redis
Mục đích chính Cache với some durability Database với Redis-compatible API
Use Case Caching, session storage Primary database cho ứng dụng
Data Durability Limited (AOF persistence) Full durability với Multi-AZ transaction log
Recovery Backup/restore Fast recovery từ transaction log
Performance High performance caching Ultra-fast performance (160M+ requests/sec)

Kết luận: Redis được thiết kế như một cache, trong khi MemoryDB for Redis được thiết kế như một cơ sở dữ liệu chính với khả năng tương thích Redis.

Hiệu năng vượt trội:
