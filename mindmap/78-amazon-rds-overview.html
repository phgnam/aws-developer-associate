Mind Map: AWS RDS (Relational Database Service)

Tổng quan về dịch vụ cơ sở dữ liệu quan hệ quản lý của AWS

AWS RDS

Relational Database Service

Định nghĩa

RDS là dịch vụ cơ sở dữ liệu quan hệ được quản lý bởi AWS dành cho các database sử dụng SQL làm ngôn ngữ truy vấn.

SQL (Structured Query Language) là ngôn ngữ chuẩn được sử dụng để tương tác với các database quan hệ.

RDS cho phép bạn tạo, vận hành và mở rộng database quan hệ trên đám mây mà không phải lo lắng về các tác vụ quản lý cơ bản.

Khái niệm Storage Auto Scaling

Tính năng tự động mở rộng lưu trữ cho RDS khi sắp hết dung lượng mà không cần can thiệp thủ công.

Cho phép ứng dụng hoạt động liên tục mà không bị gián đoạn do hết dung lượng lưu trữ.
